# SY25-26 Portfolio
## CS3: Web Development
-------------------------------------------------
1.**First Quarter** starts the school year with a review of programming concepts learning from CS2 and then transition into the use of JS as the default scripting language web browsers. Starting with the discussion of a scripting language will hopefully lay the programming foundation in making websites interactive.

2. **Second Quarter** will focus on the development of web sites using HTML, CSS and JS.  JavaScript will be used to insert interactivity into the site.  Second Quarter is also the start of a web site project that will be developed until the end of the fourth quarter.
   
4. **Third Quarter** focuss on the use of JS Object and HTML Forms and LocalStorage and updating of the website project started in the 2nd Qtr to include HTML Forms to capture users' data and do simple manipulation.
   
6. **Fourth Quarter** will be discussions on simulating CRUD with Objects saved in the LocalStorage and some front-end API
